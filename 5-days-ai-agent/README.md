# 5-Day AI Agents Intensive Course with Google

This repository folder serves as a centralized knowledge base for the **5-Day AI Agents: Intensive Vibe Coding Course** (hosted by Kaggle in partnership with Google, June 15–19, 2026). It contains summaries of core concepts, coding workflows, architectural patterns, and official course materials.

---

## Course Curriculum Overview

*   **[Day 1: Introduction to Agents & Vibe Coding](file:///Users/minhpd/Code/Technical/competitive%20programming/sample/5-days-ai-agent/day1_intro.md)**: Agentic vs. traditional LLM architectures, the "Vibe Coding" Software Development Life Cycle (SDLC), setting up the Google Agent Development Kit (ADK), and basic agent prototyping.
*   **[Day 2: Tools & Interoperability (MCP & A2A)](file:///Users/minhpd/Code/Technical/competitive%20programming/sample/5-days-ai-agent/day2_tools.md)**: Adding tools via API, standardizing toolsets with Model Context Protocol (MCP), Agent-to-Agent (A2A) communication, and Generative UI (A2UI).
*   **[Day 3: Context Engineering (Sessions & Memory)](file:///Users/minhpd/Code/Technical/competitive%20programming/sample/5-days-ai-agent/day3_memory.md)**: Managing conversational state, context windows, short-term session history, and persistent long-term Memory Banks.
*   **[Day 4: Agent Quality & Evaluation](file:///Users/minhpd/Code/Technical/competitive%20programming/sample/5-days-ai-agent/day4_quality.md)**: Observability, logging, tracing, local vs. cloud evaluations, LLM-as-a-judge metrics (task success, tool quality, hallucination), and prompt optimization.
*   **[Day 5: Prototype to Production](file:///Users/minhpd/Code/Technical/competitive%20programming/sample/5-days-ai-agent/day5_production.md)**: Moving from CLI/local play to production deployment targets (Agent Runtime, Cloud Run, GKE), multi-agent graph workflows (ADK 2.0), and the Capstone Project requirements.
